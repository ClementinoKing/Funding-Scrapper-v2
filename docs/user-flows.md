# User Flows

## Applicant Flow
1. Sign up and verify account.
2. Log in and complete profile fields used by matching engine.
3. Review dashboard KPIs and recommended actions.
4. Browse approved opportunities and open program details.
5. View match results with reasons and blockers.
6. Receive email notifications for new qualifying opportunities and deadline alerts.

## Admin Flow
1. Log in with admin role.
2. Check scraper run health and failed/partial runs.
3. Review low-confidence or pending program records.
4. Edit and approve/reject program data.
5. Review manual intervention queue and unresolved unmatched cases.
6. Tune matching weights and trigger reprocessing.
